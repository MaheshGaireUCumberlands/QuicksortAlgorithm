# Quicksort Algorithm: Implementation, Analysis, and Randomization

## Overview
This repository contains Python implementations of deterministic and randomized Quicksort, along with empirical analysis comparing their performance on different input types (random, sorted, reverse-sorted).

## Repository Structure
/code/
  - deterministic_quicksort.py
  - randomized_quicksort.py
  - empirical_tests.py

/documentation/
  - report.pdf (or .md) with analysis and performance discussion

## Running the Code
1. Navigate to `/code/`.
2. Run deterministic or randomized Quicksort scripts:
   ```bash
   python deterministic_quicksort.py
   python randomized_quicksort.py
